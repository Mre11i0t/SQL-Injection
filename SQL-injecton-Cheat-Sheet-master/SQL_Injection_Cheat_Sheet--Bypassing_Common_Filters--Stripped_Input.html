Stripped Input

Some input validation routines employ a simple blacklist and either block or remove any supplied data that appears on this list.
In this instance, you should try looking for common defects in validation and canonicalization mechanisms.

For example, if the SELECT keyword is being blocked or removed, you can try the following bypasses:
